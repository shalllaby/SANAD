# 🌙 مشروع سند (SANAD) - منصة الرعاية الذكية الشاملة
智能老年人关怀平台 | AI-Powered Elder Care Platform

**سند (SANAD)** هو نظام برمجى متطور مصمم خصيصاً لتوفير مراقبة ذكية وشاملة لكبار السن. يجمع المشروع بين أحدث تقنيات **الذكاء الاصطناعي (Computer Vision)** وحساسات إنترنت الأشياء (IoT) لضمان سلامة كبار السن وربطهم بذويهم (المراقبين) لحظة بلحظة.

---

## 🏗️ هندسة النظام (System Architecture)

يعتمد مشروع "سند" على بنية موزعة (Microservices) لضمان الكفاءة:

1.  **💻 الواجهة الأمامية (Modern Frontend):**
    *   تصميم عصري بنمط **Nebula Core (Dark Mode)** مع تأثيرات الـ Glassmorphism.
    *   **لوحة تحكم المراقب (Monitor Dashboard):** عرض تحليلات شاملة، خرائط حية، وتنبيهات فورية.
    *   **لوحة تحكم المسن (Elder Dashboard):** واجهة مبسطة جداً مع أزرار طوارئ ودردشة.

2.  **⚙️ الخادم المركزي (Node.js Backend):**
    *   بني بـ **Express.js** مع محرك **Prisma ORM**.
    *   إدارة المستخدمين، التوثيق (JWT)، الربط بين الحسابات، وجدولة التذكيرات.

3.  **🧠 محركات الذكاء الاصطناعي (AI Microservices):**
    *   **FastAPI Framework:** تشغيل موديلات الذكاء الاصطناعي كخدمات منفصلة.
    *   **Mood Detection:** تحليل تعبيرات الوجه (6 حالات: سعيد، حزين، غاضب، متفاجئ، طبيعي، مشمئز).
    *   **Fire & Smoke Detection:** كشف الحرائق والادخنة عبر الكاميرات.
    *   **Fall Detection:** كشف حالات السقوط (محاكاة).

## 🛠️ الميزات الأساسية (Core Features)

### 🏥 المراقبة الصحية (Health Monitoring)
*   تتبع نبض القلب (BPM)، مستوى الأكسجين، وضغط الدم.
*   رسوم بيانية تفاعلية (Chart.js) تعرض تذبذب الحالة الصحية أسبوعياً.

### 🛡️ الأمان والذكاء الاصطناعي (AI Security)
*   **تنبيهات فورية:** إرسال إنذارات عند اكتشاف حريق أو سقوط أو حالة مزاجية سيئة (Risk Alert).
*   **AI Lab:** مختبر متكامل لاختبار الموديلات عبر فيديوهات تجريبية.
*   **دليل مرئي (Video Evidence):** عرض لقطة الفيديو التي تسببت في التنبيه لضمان التحقق.

### 📅 إدارة المهام والتذكيرات
*   جدولة الأدوية، الوجبات، والمواعيد الطبية.
*   نظام تكرار ذكي (يومي، أسبوعي، شهري).
*   تقارير الالتزام (عرض المهمات المكتملة والمفقودة).

### 📍 التتبع الجغرافي (Live Tracking)
*   خرائط تفاعلية (Leaflet.js) تعرض موقع المسن بدقة.
*   تنبيهات عند الخروج من المنطقة الآمنة (Geofencing - مخطط له).

### 💬 الدردشة والمساعدة الذكية (AI Voice Chat)
*   دردشة مباشرة بين المراقب والمسن.
*   **نظام صوتي متكامل:** تحويل النص إلى كلام (TTS) وتحدث المسن مباشرة (STT).
*   **مشغل صوتي احترافي:** واجهة تشبه "واتساب" تتيح للمسن الاستماع والتحكم في الرسائل الصوتية.
*   **حالة السيرفر اللحظية:** مؤشر ذكي يوضح للمستخدم إذا كان المساعد "سند" متصلاً أم لا.

## 🎙️ تحديثات نظام الدردشة الصوتية (Latest AI Chat Updates)

تم تطوير نظام الدردشة ليكون "صوتياً بالكامل" ليناسب احتياجات كبار السن:

1.  **بصمات صوتية للمسّن (User Voice Notes):** أصبح بإمكان المسن تسجيل صوته الحقيقي وإرساله، مع واجهة تفاعلية للتحكم في المقطع (تشغيل، إيقاف، تقديم).
2.  **تحكم يدوي كامل (Manual Recording):** المسن هو من يقرر متى يبدأ ومتى ينتهي التسجيل، لضمان عدم انقطاع الرسالة أثناء صمته للتفكير.
3.  **خدعة حساب الوقت (Duration Fix):** تم حل مشكلة ظهور وقت المقطع `0:00` عبر تقنية "Force Duration" لضمان عرض وقت التسجيل بدقة في كل المتصفحات.
4.  **ربط الموديلات الخارجية:** ربط النظام بسيرفر خارجي (CareCompanion AI via ngrok) لمعالجة اللغة الطبيعية (Llama-3.3) وإنتاج الأصوات (TTS).

## 🚀 كيفية التشغيل (Setup)

### 1- تشغيل الخادم (Backend):
```bash
cd server
npm install
npx prisma db push  # لتهيئة قاعدة البيانات
npm run dev
```

### 2- ربط الذكاء الاصطناعي (AI Connect):
تأكد من تشغيل سيرفر `ngrok` الخاص بالذكاء الاصطناعي وتحديث الرابط في ملفات `client/js/chat.js` و `server/src/routes/chat.js`.
